# 🚀 YnotB Portfolio

A modern full-stack portfolio and software agency website built with the MERN Stack.

Live Demo:
https://ynotb-portfolio.onrender.com

---

# 🌟 Overview

YnotB Portfolio is a professional software agency portfolio designed to showcase projects, services, team members, and collect client inquiries.

The application includes a secure admin dashboard that allows authorized users to manage website content dynamically through MongoDB Atlas.

- Responsive modern UI

# ⚙ Installation

Clone repository

```bash
git clone https://github.com/YourUsername/ynotb-portfolio.git
```

Backend

```bash
cd backend
npm install
npm start
```

Frontend

```bash
cd frontend
npm install
npm run dev
```
